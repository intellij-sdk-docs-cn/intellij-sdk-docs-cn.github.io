IntelliJ Platform SDK

欢迎来到 IntelliJ Platform SDK。这是通过创建插件，自定义语言支持，或构建自定义 IDE 来扩展 IntelliJ Platform 的主要文档来源。

更新

最新的更改请参见内容更新。 在 Twitter 上关注 JBPlatform 并访问 JetBrains Platform 博客来获取最新公告。

升级你的插件到最新 platform 发行版？确保你检查了不兼容的更改以及值得注意的更改和特性。

2020.1 将迎来动态插件。我们还发布了 2020 年的 IntelliJ Platform 路线图：部分 I 部分 II

如果你的插件依赖 Java 功能并且是 2019.2 或更高版本，请确保遵循此博客文章中的步骤。

开源

这个指南是开源的，并且以 Apache 2.0 协议开源。源代码（作为 Markdown）被托管在 GitHub 上。 请参见 CONTRIBUTING.md 来获取有关如何在本地上托管文档与进行贡献的详细信息。

如果你在这个指南中遇到了 bug，或需要缺失内容的帮助，请参见获取帮助。
